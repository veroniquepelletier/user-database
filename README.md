user-database
=============

Test with nodejs and mongodb for a user database. 


Project overview
--------------


**UI**

Grunt, Yeoman, Bower, Angular, jquery

**Server**

Nodejs

**Database**

MongoDB


Project installation
--------------


install grunt

  `npm install -g grunt-cli`

install bower

  `npm install -g grunt-cli bower`

install package

  `npm install`
  
  `bower install`

Run the project
--------------

  `grunt serve`
  
  `node server`
